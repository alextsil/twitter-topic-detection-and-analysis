Tasks & Techniques

Event detection: The top 4 ranked hashtags, their occurrences and the tweet that qualified as an event by our system will be presented. A combination of approaches was used for event detection: Terms were considered as living organisms, allowing us to sample a set of tweets as when each term was "most alive", while combining user reputation metrics to select a tweet from a reputable source.

Sentiment analysis: An Opinion-Mining tool implemented for the needs of a previous course was used to predict the polarity of the tweets related to the emerging event. A SVM classifier was trained with 25k labeled movie reviews, from the publicly known labeled dataset of imdb, in order to predict the polarity of the unlabeled dataset of the collected tweets.
